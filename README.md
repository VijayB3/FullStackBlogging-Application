# 📝 FullStack Blogging Application

A full-stack blogging web application that allows users to create, read, update, and delete blog posts with authentication and user management features.

---

## 🚀 Features.

- 🔐 User Authentication (Login / Signup)
- ✍️ Create, Edit, and Delete Blog Posts
- 📖 View all blogs and individual blog details
- 👤 User Profile Management
- 🔍 Search / Filter blogs (if implemented)
- ⚡ Responsive UI for better user experience

---

## 🛠️ Tech Stack

### Frontend
- React.js
- HTML5, CSS3
- Axios / Fetch API

### Backend
- Node.js
- Express.js

### Database
- MongoDB (Mongoose)

### Tools & Technologies
- Git & GitHub
- RESTful APIs
- JSON Web Tokens (JWT) (if used)
